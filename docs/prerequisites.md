# Pre-Requisites

## Software Requirements

* Python 3.10+
* Git
* Visual Studio Code

## Libraries / Frameworks

* FastAPI
* Streamlit
* Librosa
* OpenAI Whisper
* Sentence-BERT
* ReportLab
* NumPy
* Matplotlib
* PyTorch

## Hardware Requirements

* Processor: Intel i3/i5 or higher
* RAM: Minimum 4GB (8GB recommended)
* Storage: 10GB free disk space
* Internet connection required

## Purpose

These tools and libraries are required for application development, testing, model execution, and deployment.
